== Description ==

Customizable widget and shortcode to display tag cloud anywhere you want. You can use it as standard tag cloud widget, as shortcode in any post or page, or both ways. Tons of options allow you to fine tune cloud layout to your needs.

**STANDARD FEATURES**

- Display list of post, category or link category tags.
- Define number of tags to be displayed.
- Set custom tag separator.
- Select tag taxonomy: posts, categories or link categories.
- Choose cloud mode: shortcode only, widget only or both.

**PREMIUM FEATURES**

- Define smallest and largest tag size.
- Select size unit: px, pt, em or percent.
- Choose layout: standard flat cloud or unordered list.
- Order tags by name or count.
- Sort tags ascending, descending or randomize.
- Define list of tags to exclude from cloud or include only.

You can purchase Premium version <a href="http://thekrotek.com/wordpress-extensions/tag-cloud">here</a>.

== Installation ==

1. Unpack and upload all files to "/wp-content/plugins/plugin-name" directory. Or simply install plugin through Plugins page.
2. Activate extension on "Plugins" page.
3. Setup plugin according to your liking.

If you want to display tag cloud with shortcode, add [tagcloud id=XXX] in your post or page, replacing XXX with desired widget number.
